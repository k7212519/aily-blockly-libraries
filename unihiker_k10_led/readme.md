# K10 RGB LED

UNIHIKER K10 板载 RGB LED 控制库，支持亮度和颜色设置。

## 库信息

| 字段 | 值 |
|------|-----|
| 包名 | @aily-project/lib-unihiker-k10-led |
| 版本 | 0.1.0 |
| 作者 | DFRobot |
| 来源 | https://github.com/DFRobot/UNIHIKER_K10 |
| 许可证 | MIT |

## 支持的开发板

UNIHIKER K10 (DFR0992)

## 说明

K10 开发板板载 3 颗 WS2812 RGB LED，可独立控制每颗灯的颜色，也可统一设置亮度。

## 快速开始

1. 使用「设置RGB灯亮度」积木调整亮度
2. 使用「设置RGB灯颜色」积木设置灯的颜色
