## 支持的开发板

UNIHIKER K10 (DFR0992)

## 说明

K10 开发板集成语音识别和语音合成（TTS）功能，支持中英文语音唤醒、自定义命令词识别和语音播报。可设置播报速度。

## 快速开始

1. 使用「初始化语音识别」积木初始化 ASR 模块
2. 使用「添加语音命令」积木注册自定义命令词
3. 在循环中检测唤醒状态和命令词识别结果
