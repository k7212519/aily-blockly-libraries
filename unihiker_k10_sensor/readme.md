# K10传感器

UNIHIKER K10 板载传感器库，支持加速度计和光线/声音强度检测。

## 库信息

| 字段 | 值 |
|------|-----|
| 包名 | @aily-project/lib-unihiker-k10-sensor |
| 版本 | 0.1.0 |
| 作者 | DFRobot |
| 来源 | https://github.com/DFRobot/UNIHIKER_K10 |
| 许可证 | MIT |

## 支持的开发板

UNIHIKER K10 (DFR0992)

## 说明

K10 开发板集成三轴加速度计（SC7A20H）和光线传感器（LTR303ALS），可获取 X/Y/Z 轴加速度数据、光线强度和运动强度值。

## 快速开始

1. 直接使用「获取加速度计」积木读取 X/Y/Z 轴数据
2. 使用「获取光线/声音强度」积木读取环境传感器数据
