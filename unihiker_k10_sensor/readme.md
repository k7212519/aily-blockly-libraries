# K10传感器

UNIHIKER K10 板载传感器库，支持加速度计、光线/声音强度检测以及板载 AHT20 温湿度传感器读取。

## 支持的开发板

UNIHIKER K10 (DFR0992)

## 说明

K10 开发板集成三轴加速度计（SC7A20H）、光线传感器（LTR303ALS）以及 AHT20 温湿度传感器（I2C 地址 0x38），可获取 X/Y/Z 轴加速度、光线/声音强度，以及环境温度（℃/℉）和相对湿度（%RH）。

## 快速开始

1. 使用「获取加速度计」积木读取 X/Y/Z 轴数据
2. 使用「获取光线/声音强度」积木读取环境传感器数据
3. 在循环中先调用「AHT20 启动测量」积木，当返回 true 时再使用「AHT20 获取温度 / 获取相对湿度」积木读取数据
