# K10音乐/音频

UNIHIKER K10 音乐与音频库，支持内置音乐播放、音调播放、TF 卡录音和播放。

## 支持的开发板

UNIHIKER K10 (DFR0992)

## 说明

K10 开发板集成 I2S 音频输出，支持播放多首内置音乐（生日快乐、小星星等）、自定义频率音调播放、TF 卡 WAV 音频录制和播放。

## 快速开始

1. 使用「播放内置音乐」积木播放预设曲目
2. 使用「播放音调」积木播放自定义频率的声音
3. 使用「录音/播放TF卡音频」积木进行音频录放
