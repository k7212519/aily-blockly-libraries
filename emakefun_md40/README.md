# MD40 四路电机驱动库

MD40 是一款通过 I2C 通信的四路电机驱动模块，支持直流模式（PWM控制）和编码器模式（速度/位置闭环控制）。

## 功能概述

- 支持4路电机独立控制
- **直流模式**：通过 PWM 占空比直接控制电机转速和方向
- **编码器模式**：支持速度闭环、位置闭环控制
- I2C 通信，默认地址 0x16
- 支持速度 PID 和位置 PID 参数调节

## 块说明

### 初始化

| 块 | 说明 |
|---|---|
| 初始化MD40电机驱动 | 创建 MD40 对象，设置 I2C 地址，自动初始化 Wire 和模块 |

### 直流模式

| 块 | 说明 |
|---|---|
| 设置电机为直流模式 | 将指定电机切换为直流模式 |
| 以PWM占空比运转 | 以 -1023~1023 的 PWM 值驱动电机，正数正转，负数反转 |

### 编码器模式

| 块 | 说明 |
|---|---|
| 设置电机为编码器模式 | 配置 PPR（每转脉冲数）、减速比和相位关系 |
| 设置速度PID | 设置速度环 P、I、D 参数 |
| 设置位置PID | 设置位置环 P、I、D 参数 |
| 以速度RPM运转 | 以指定 RPM 速度运行电机 |
| 转动到位置 | 转动到绝对角度位置 |
| 相对转动 | 相对当前位置转动指定角度 |

### 控制

| 块 | 说明 |
|---|---|
| 停止电机 | 停止指定电机 |
| 重置电机 | 重置电机到初始状态 |
| 设置位置值 | 设定电机位置值（用于校准） |
| 设置脉冲计数 | 设定编码器脉冲计数值 |

### 状态读取

| 块 | 返回值 |
|---|---|
| 当前转速(RPM) | 电机输出轴转速，正数正转，负数反转 |
| 当前位置(度) | 电机累计角度值 |
| 脉冲计数 | 编码器脉冲计数 |
| 当前PWM占空比 | -1023~1023 |
| 当前状态 | 0空闲, 1PWM运行, 2速度运行, 3位置运行中, 4已到达 |

## 使用流程

### 直流模式

1. 拖入"初始化MD40电机驱动"
2. 拖入"设置电机为直流模式"
3. 使用"以PWM占空比运转"控制电机

### 编码器速度控制

1. 拖入"初始化MD40电机驱动"
2. 拖入"设置电机为编码器模式"（PPR=12, 减速比=90）
3. 拖入"设置速度PID"（推荐 P=1.5, I=1.5, D=1.0）
4. 使用"以速度RPM运转"控制电机

### 编码器位置控制

1. 拖入"初始化MD40电机驱动"
2. 拖入"设置电机为编码器模式"
3. 拖入"设置速度PID"和"设置位置PID"（位置推荐 P=10.0, I=1.0, D=1.0）
4. 使用"转动到位置"或"相对转动"控制电机

## 注意事项

- 使用编码器模式前，必须先设置编码器模式和PID参数
- PPR（每转脉冲数）和减速比需根据实际电机参数设置
- 位置控制需要同时配置速度PID和位置PID
- 电机索引范围 0~3，对应模块上的4个电机通道
- 支持 Arduino AVR 和 ESP32 平台
