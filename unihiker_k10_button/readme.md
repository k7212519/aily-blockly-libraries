# K10按键

UNIHIKER K10 板载按键库，支持 A/B 按键轮询检测和回调。

## 库信息

| 字段 | 值 |
|------|-----|
| 包名 | @aily-project/lib-unihiker-k10-button |
| 版本 | 0.1.0 |
| 作者 | DFRobot |
| 来源 | https://github.com/DFRobot/UNIHIKER_K10 |
| 许可证 | MIT |

## 支持的开发板

UNIHIKER K10 (DFR0992)

## 说明

K10 开发板内置两个用户按键（A 和 B），支持轮询检测按键状态和注册按下/释放回调函数。

## 快速开始

1. 直接使用「K10按键被按下」积木检测按键状态
2. 或使用「当K10按键被按下时」积木注册回调处理
