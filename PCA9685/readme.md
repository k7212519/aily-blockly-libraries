# PCA9685 舵机扩展模块

Adafruit PCA9685 是一款 16 通道 12 位 PWM 舵机驱动板，通过 I2C 接口控制，可同时驱动多达 16 个舵机或 LED。

## 库信息

- **库名**: @aily-project/lib-pca9685
- **版本**: 1.0.0
- **兼容**: Arduino AVR、ESP32
- **电压**: 3.3V / 5V
- **原始库**: Adafruit_PWMServoDriver

## 功能特性

- 支持 16 通道独立 PWM 输出
- 舵机角度控制（0°~180°）
- PWM 值直接控制（0~4095）
- 微秒级脉冲控制
- 可设置 PWM 频率（最高约 1.6KHz）
- 睡眠/唤醒模式控制
- 支持多个 PCA9685 级联（不同 I2C 地址）

## 使用说明

### 接线方式

| PCA9685 | Arduino UNO | ESP32 |
|---------|-------------|-------|
| VCC | 5V | 5V |
| GND | GND | GND |
| SDA | A4 | GPIO21 |
| SCL | A5 | GPIO22 |

### 基本使用流程

1. **创建 PCA9685 对象** — 设置 I2C 地址
2. **初始化** — 调用 begin() 启动 I2C 通信
3. **设置 PWM 频率** — 舵机一般设为 50Hz
4. **控制舵机** — 设置角度或 PWM 值

### 舵机频率说明

| 舵机类型 | 推荐频率 |
|---------|---------|
| 模拟舵机 | 50 Hz |
| 数字舵机 | 50~200 Hz |
| LED 调光 | 1000~1600 Hz |

### I2C 地址设置

PCA9685 默认地址为 0x40，通过焊接背面的跳线焊盘可设置 0x40~0x7F 范围内的地址。每片 PCA9685 最多可设置 64 个不同地址，满足大规模级联需求。

### 注意事项

- 使用前请确保已安装 Adafruit_PWMServoDriver 和 Adafruit_BusIO 库
- 舵机实际脉宽范围可能因品牌不同而异，默认 SERVOMIN=150, SERVOMAX=600
- 大电流应用时请为 PCA9685 板提供独立电源（V+），不要仅依靠 VCC
